## Installation
#### Nuxtjs:

    1. clone repository  https://github.com/afitra/sv_frontend_test.git
    2. masuk ke folder hasil clone
    3. setup .env file seuai di file .env.example (sebelumnya jalankan backend terlebih dahulu)
    4. npm install
    5. npm run dev
    6. enjoy with nuxtjs
